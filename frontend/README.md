# Holiday Analyzer Frontend

React application built with Vite and TypeScript.

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Docker

```bash
# Build Docker image
docker build -t holiday-analyzer-frontend .

# Run container
docker run -p 80:80 holiday-analyzer-frontend
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Nginx (for production)
